==========
통합 패턴
==========

OpenStack 과 Kubernetes 를 결합하는 두 가지 방향을 다룹니다. OpenStack 컨트롤
플레인을 Kubernetes 위에 올리는 방향과, 반대로 Kubernetes 클러스터를 OpenStack
인프라 위에 올리는 방향으로 나뉩니다.

.. note::

   기여자 작업용 골격입니다. 각 방향의 상세는 하위 문서에서 보강해 주세요.
