====================
Neutron 기초
====================

Neutron 은 OpenStack 의 가상 네트워킹(SDN) 서비스입니다. 실습 전에 잡아 두면
좋은 핵심 구성요소와 개념을 정리합니다.

.. note::

   기여자 작업용 골격입니다. 실습 경험과 공식 문서를 바탕으로 보강해 주세요.


핵심 개념
=========

* 네트워크 / 서브넷 / 포트
* 프로바이더 네트워크 vs 테넌트(프로젝트) 네트워크
* 라우터, 플로팅 IP, 시큐리티 그룹
