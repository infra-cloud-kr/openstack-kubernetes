============================
Swift (오브젝트 스토리지)
============================

Swift 는 OpenStack 의 오브젝트 스토리지 서비스입니다. Kubernetes 생태계에서도
여러 용도로 쓰입니다.

.. note::

   기여자 작업용 골격입니다. 실제 사용 사례를 바탕으로 보강해 주세요.


주요 활용
=========

* **kOps 클러스터 state store** — 클러스터 상태 저장소
* **Glance 이미지 백엔드** — OpenStack 이미지 저장
* **애플리케이션 오브젝트 스토리지** — S3 호환 인터페이스 활용


개념
====

* 계정 / 컨테이너 / 오브젝트 구조
* 복제(replication)와 내구성
* S3 호환 API
