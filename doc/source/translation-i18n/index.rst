================
번역 · 국제화
================

번역과 국제화(i18n/l10n)를 다룹니다. 두 오픈소스 기술을 직접 이해한 상태에서
공식 문서를 100% 한글화하고, 번역 플랫폼과 AI 보조 번역 워크플로우를
정리합니다.

먼저 :doc:`localization-communities` 에서 OpenStack 과 Kubernetes 양쪽의
한글화 공동체(OpenStack I18n SIG 와 Kubernetes SIG Docs 현지화 하위
프로젝트)가 어떻게 구성되어 있고 본 프로젝트가 그 사이 어디에 위치하는지
조망한 뒤, 각 공동체의 워크플로우와 도구를 차례로 살펴봅니다.
