========================
Kubernetes 현지화
========================

Kubernetes 문서는 언어별 디렉터리와 SIG Docs Localization 하위 그룹을 통해
현지화됩니다. OpenStack i18n 과는 도구·프로세스가 다릅니다.

.. note::

   기여자 작업용 골격입니다. 한국어(ko) 현지화 팀의 가이드를 확인하며
   보강해 주세요.


핵심 출처
=========

* Localizing Kubernetes Documentation —
  https://kubernetes.io/docs/contribute/localization/


OpenStack 과의 차이
===================

* **방식**: ``.po`` 기반 메시지 번역(OpenStack) vs 언어별 Markdown 파일
  복제·번역(Kubernetes)
* **도구**: Weblate 등 플랫폼(OpenStack) vs GitHub PR 중심(Kubernetes)
* 자세한 도구 비교는 :doc:`../documentation/kubernetes-doc-conventions`
