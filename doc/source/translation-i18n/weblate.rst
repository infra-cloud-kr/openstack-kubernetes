=======
Weblate
=======

`Weblate <https://weblate.org/>`_ 는 웹 기반 번역 플랫폼입니다. 본 프로젝트는
번역 워크플로우 구성 경험을 쌓기 위해 Weblate 활용을 다룹니다.

.. note::

   기여자 작업용 골격입니다. 실제 인스턴스 구성/연동 경험을 바탕으로
   보강해 주세요.


왜 Weblate 인가
===============

* 웹 기반의 직관적인 번역 인터페이스
* VCS(Git) 연동을 통한 ``.po`` 동기화
* 번역 메모리, 용어집, 품질 검사 기능


기본 흐름
=========

#. Git 저장소와 Weblate 컴포넌트 연결
#. ``.pot``/``.po`` 파일 등록
#. 번역 진행 및 리뷰
#. 번역 결과를 Git 으로 커밋/푸시


연동 시 고려사항
================

* 저장소 권한과 브랜치 전략
* 충돌 해결과 동기화 주기
* 용어집/번역 메모리 공유
